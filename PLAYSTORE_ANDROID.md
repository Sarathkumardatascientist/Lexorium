# Lexorium Android / Play Store

This repository now includes an Android shell app in `android-playstore/` that wraps the live Lexorium deployment.

What it includes:
- Lexorium-branded Android app shell
- Secure WebView pointed to `https://lexoriumai.com`
- file and image upload support
- microphone and camera permission handling
- download handling through Android `DownloadManager`
- pull-to-refresh
- branded splash/logo usage
- Play-target-ready SDK settings (`targetSdk 35`)

Open in Android Studio:
1. Open the `android-playstore/` folder in Android Studio.
2. Let Android Studio sync Gradle and install the Android SDK if prompted.
3. If you move Lexorium to a new domain later, update `LEXORIUM_BASE_URL` in `android-playstore/app/build.gradle.kts`.
4. Build a signed Android App Bundle (`Build > Generate Signed Bundle / APK > Android App Bundle`).

Before Play submission:
1. Replace the launcher icon set if you want optimized Play assets beyond the shared website logo.
2. Test sign-in, file upload, voice permissions, and payment on a real Android device.
3. Create the Play Console listing, screenshots, privacy policy link, and Data safety answers.
4. Upload the signed `.aab` generated by Android Studio.

Important:
- This is a native Android shell around the live Lexorium deployment, not a fully offline app.
- If you later move to a custom production domain, update the base URL and network security config before publishing an update.
